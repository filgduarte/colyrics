Antes de iniciar a implementação, leia e utilize como contexto todos os arquivos relevantes dentro da pasta:

`/docs/ai/`

Regras:

* Considere os arquivos dessa pasta como documentação oficial do projeto.
* Priorize as convenções, arquiteturas e padrões definidos nesses documentos.
* Utilize os prompts e especificações existentes como referência para manter consistência.
* Não contradiga decisões previamente documentadas.
* Caso alguma instrução entre em conflito, siga esta prioridade:

  1. architecture
  2. conventions
  3. specs
  4. prompts

Arquivos importantes:

* `/docs/ai/README.md`
* `/docs/ai/conventions/*`
* `/docs/ai/architecture/*`
* `/docs/ai/specs/*`

Após analisar os arquivos, implemente a funcionalidade solicitada.
